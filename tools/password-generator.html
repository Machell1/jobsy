How to Use This Password Generator

Adjust the length slider to set your desired password length (4 to 64 characters). Toggle the character types you want to include: uppercase letters, lowercase letters, digits, and special symbols. Click "Generate" for a new password.

This generator uses your browser's built-in cryptographic random number generator (crypto.getRandomValues), which provides true randomness suitable for security-critical applications. The strength meter shows the password entropy in bits.

A strong password should have at least 60 bits of entropy. For maximum security, use 16+ characters with all character types enabled. Your passwords are generated entirely in your browser and are never transmitted over the internet.
