How to Use This Unit Converter

Select a category (length, weight, temperature, speed, or data storage), then choose the units to convert from and to. Enter a value and see the result instantly. Use the swap button to quickly reverse the conversion direction.

The converter supports common units in each category: kilometers, miles, meters, feet, inches for length; kilograms, pounds, ounces for weight; Celsius, Fahrenheit, Kelvin for temperature; and bytes through terabytes for data storage.

All conversions use standard conversion factors and are calculated in real time in your browser. Results are displayed with full precision so you can use them confidently in calculations, schoolwork, or professional applications.
