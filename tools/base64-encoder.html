How to Use This Base64 Encoder

Enter your text in the input area and click "Encode" to convert it to Base64 format. To reverse the process, paste a Base64 string and click "Decode". The "Swap" button lets you quickly move the output back to the input for chaining operations.

You can also drag and drop any file onto the drop zone to get its Base64 representation. This is useful for embedding small images, fonts, or other binary data directly in HTML or CSS files.

This tool fully supports Unicode text, including emojis and characters from any language. All encoding and decoding happens in your browser.
