# Jobsy - Jamaica's Service Marketplace

Jobsy connects Jamaicans with trusted local service providers across all 14 parishes. From home repair and cleaning to beauty services and events, find the right professional for every job.

## Features

- **Service Directory** — Browse 10+ categories of local services
- **Island-wide Coverage** — Service providers across all 14 parishes
- **Telegram Deal Bot** — @JobsyDealBot for real-time deal alerts
- **Affiliate Integration** — Amazon, eBay, CJ, AliExpress partnerships
- **Mobile-first Design** — Responsive, fast-loading website

## Tech Stack

- **Website**: HTML, CSS, JavaScript (GitHub Pages)
- **Telegram Bot**: Python (Railway)
- **Backend**: Microservices architecture (Node.js/Python)
- **Domain**: [www.jobsyja.com](https://www.jobsyja.com)

## Service Categories

- Home Repair (Plumbing, Electrical, Pest Control)
